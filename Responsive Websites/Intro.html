====================================== In general, HTML elements are divided into 2 categories 1. Block-level Elements -:--- Always starts in new line and takes Full width ( occupies entire hori space of its parent element ) h1, p, div 2. Inline Elements -:-- Do not start in new line and only takes up as much width as necessary ( button, img, a ) ------------------------------------------------------- CSS Box Properties -:-- Margin -:-- We can align HTML Block-level elements horizontally using CSS margin property margin: auto; If we specify CSS text-align prop to Block-level element, it aligns the text or HTML Inline elements inside it margin-right: auto; ---- Takes up all availabe space on right and element gets aligned to left margin-left: auto; ----- Takes up all availabe space on left and element gets aligned to right *** Bootstrap Utility ---- m-auto, ml-auto, mr-auto ============================================================================================== CSS Selectors and Inheritance : Selectors are used to select HTML elements that we want to style. Simple Selectors -:--- Class, Id, Type (tag name), Attribute, Universal,Pseudo-class selector Compound and Complex Selectors and many more Class Selector : selects all HTML elements that have a given CSS class selector as their class attribute value .paragraph { color: blue; } . (dot) notation, can be more than 1 HTML element with same class name in document ID Selector : CSS ID selector selects the HTML based on its ID attribute value #populationPara { color: red; } # (hash) notation , Only 1 HTML element with given ID in entire HTML document. Type (tag name) Selector : Selects all HTML elements based on their tag names h1 { color: red; } =================================================================================================== CSS Inheritance : Mechanism through which the value of CSS Properties is passed on from parent's to child elements CSS Properties are divided into 2 categories : Inherited Properties -:--- CSS properties applied to parent element are inherited by child elements font-family, font-style, font-weight, text-align, list-style-type, color ------ Non-inherited Properties -:-- CSS properties applied to parent element are not inherited by child elements width, height, margin, padding, border-style(width, color, radius), background-image(color, size), text-decoration ------------------- ------------------------------------------------------- Specificity : How the browsers decide which CSS property values are most relevant to an HTML element and, will be applied Lowest to Highest Order by Specificity: Type (tag name) < Class Selector < ID Selector ID Selector is more specific compared to Class Selector which is more specific compared to Type Selector ***** Doesn't overwrite the entire CSS but only overwrites the CSS properties that are same ****** Inline Styles : Inline styles are applied directly to HTML element.
